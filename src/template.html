Projects

RoboCards

RoboCards is a simple site that uses search to filter through a list of Robot Cards. The project was bootstrapped using Create React App. The List of Card Names is supplied through an API called JSON Placeholder API and Robot Images are supplied for every Unique Robot Name using an API called Robo Hash. The App is also styled using tachyons which is a simple CSS toolkit for quickly building and designing new UI without writing actual CSS. In an Earlier Version of this Project, I used Redux Thunk for state management (mostly just to test out the technology) but later on removed it to avoid over engineering. The current method used to handle state simply involves a Constructor Function housing a this.state Object. The App was deployed using Netlify which is a great platform for deploying modern static websites.

Smartbrain

Smartbrain is a simple site for detecting faces in pictures. All you have to do is click signin and copy the image link into the input box and click detect. If there's a face in the picture, the app will simply put a box around the face. This project was bootstrapped using Create React App and makes use of the Clarifai API for face detection. It also uses tachyons, tilt.js and particles.js for styling and a little fancy interactivity. SignIn and Registration forms where added to this project without authentication to test out Routing which was handled simply using a route state instead of React Router to avoid over engineering. Future Additions to this project will feature a Database, User Authentication, a backend using NodeJs, an Upload functionality and further experimentation with the Clarifai API to detect Food, Color and other stuff. This app was also deployed using Netlify.

Quiz App

This is a Sleek Website built using React and TypeScript. The project was styled using styled components which is a CSS-In-JS tool and a great alternative to tachyons (particularly if you want more control over your styles) as it allows you to make custom styles for your components and brings greater interactivity to your styles without the added complexity. The Project uses an API called Open Trivia to source the random quiz questions. Fonts used in this project are Catamaran and Fascinate inline from Google Fonts and the App is also deployed using Netlify.
