Mini Trello

This is a slimmed down version of the popular web-based, Kanban-style, list-making application Trello. Tasks can be added under the 3 sections by clicking add new task and they can be moved from To-do to In-progress to Done by simply double clicking the task. It was built with React and TypeScript, deployed on Netlify and uses the browsers local storage to persist Data.
