About me

At Decagon, I was able to hone my skills and learn a lot of things, and the most important of them all was learning how to learn. Right now I'm a full stack web developer capable of leveraging JavaScript and TypeScript to Build Full Stack Applications. I'm now officially a TypeScript fan and I'm currently using it along with React, GraphQL, Node.js, Express and Postgres to build a Reddit Clone. Asides coding I've also been fascinated by how companies apply technology to solve problems and this has made me study them and the products they build. Occasionally I write about them on LinkedIn and Medium but due to my recent commitments I haven't been able to write in months. I look forward to working with teams solving interesting problems at scale and having a little more time to write.

Projects

Mini Trello

This is a slimmed down version of the popular web-based, Kanban-style, list-making application Trello. Tasks can be added under the 3 sections by clicking add new task and they can be moved from To-do to In-progress to Done by simply double clicking the task. It was built with React and TypeScript, deployed on Netlify and uses the browsers local storage to persist Data.

WhatsApp Clone

A basic whatsapp clone supporting one-to-one messaging built with Next.js and Firebase. Elements of Server Side Rendering and Static Site Generation are used within the app. User authentication is provided by Google Auth, User data is stored on Firestore and Web App was deployed on Vercel.

Todo App

A Basic Todo App that supports adding todos, editing exiting todos, deleting todos, copying todos and searching through todos. The App leverages all the major full stack components with the Front End handled by vanilla HTML, CSS and JavaScript and the Backend handled by Node.js (TypeScript), Express.js and MongoDB. The Web App was also deployed to Heroku.
